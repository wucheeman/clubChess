Create a README
===============
[] Add a `README.md` to your repository describing the project. Review these:
  * [About READMEs](https://help.github.com/articles/about-readmes/)
  * [Mastering Markdown](https://guides.github.com/features/mastering-markdown/)
[] Add animation, e.g. a GIF. This is really impressive.

Add To GitHub Pages or Heroku
=============================
[] Create a GitHub Pages page or Heroku app for the project
[] Confirm Github Pages page/Heroku app is accessible from the Internet
[] Complete full test round on GitHub Pages page/Heroku app and confirm no issues
[] Add the app to your portfolio via an active link and update GitHub.
[] Confirm updated portfolio in GitHub Pages is accessible and no issues
